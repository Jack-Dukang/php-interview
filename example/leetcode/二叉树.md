# 二叉树常见总结
二叉树 主要是递归
涉及到个数就递归+回溯


## 第一类 遍历相关
本类型涉及到前/中/后序遍历，以及层序遍历

### 1.1 前/中/后序遍历
```php
94. 二叉树的中序遍历
https://leetcode-cn.com/problems/binary-tree-inorder-traversal/

<?php

    /**
     * @param TreeNode $root
     * @return Integer[]
     */
    function inorderTraversal($root){
        $result = [];
        $this->getVal($root,$result);
        return $result;
    }

    function getVal($node,&$result){
        if ($node == null){
            return null;
        }

        $this->getVal($node->left,$result);
        $result[] = $node->val;
        $this->getVal($node->right,$result);
    }

```

// leetcode 102的示例
class Solution {
    public $result = [];
    /**
     * @param TreeNode $root
     * @return Integer[][]
     */
    function levelOrder($root) {
        if($root == null){
            return $this->result;
        }
        $this->dfs($root,0);
        return $this->result;
    }

    function dfs($root,$level){
        if($root === null){
            return ;
        }
        $this->result[$level][] = $root->val;
        $this->dfs($root->left,$level+1);
        $this->dfs($root->right,$level+1);
    }
}

## 二、二叉树建立
### 2.1、二叉树重建
```php
105. 从前序与中序遍历序列构造二叉树
根据一棵树的前序遍历与中序遍历构造二叉树。
https://leetcode-cn.com/problems/construct-binary-tree-from-preorder-and-inorder-traversal/

解题思路：
通过递归的方式，前序遍历是根左右，中序遍历是左根右，然后根据根可以知道左边的元素，不断的递归即可

参考答案：
<?php

class TreeNode{
    public $left = null;
    public $right = null;
    public $val = null;
    public function __construct($val){
        $this->val = $val;
    }
}

    /**
     * @param Integer[] $preorder
     * @param Integer[] $inorder
     * @return TreeNode
     */
    function buildTree($preorder, $inorder) {
        if($preorder && $inorder){
            $node = new TreeNode($preorder[0]);
            $index = array_search($preorder[0],$inorder);
            $node->left = $this->buildTree(array_slice($preorder,1,$index),array_slice($inorder,0,$index));
            $node->right = $this->buildTree(array_slice($preorder,$index+1),array_slice($inorder,$index+1)  );
            return $node;
        }
    }
```
### 2.2、合并二叉树
```php
617. 合并二叉树
https://leetcode-cn.com/problems/merge-two-binary-trees/
思路：递归

<?php

class Solution {

    /**
     * @param TreeNode $t1
     * @param TreeNode $t2
     * @return TreeNode
     */
    function mergeTrees($t1, $t2) {
        if($t1 == null){
            return $t2;
        }
        if($t2 == null){
            return $t1;
        }
        $t1->val = $t1->val + $t2->val;
        $t1->left = $this->mergeTrees($t1->left,$t2->left);
        $t1->right = $this->mergeTrees($t1->right,$t2->right);
        return $t1;
    }
}

```

## 三、验证二叉数相关

### 3.1 验证二叉搜索树
```php
98. 验证二叉搜索树
给定一个二叉树，判断其是否是一个有效的二叉搜索树。

假设一个二叉搜索树具有如下特征：

节点的左子树只包含小于当前节点的数。
节点的右子树只包含大于当前节点的数。
所有左子树和右子树自身必须也是二叉搜索树。

链接：https://leetcode-cn.com/problems/validate-binary-search-tree

<?php

class Solution {
    function isValidBST($root){
        return $this->task($root,null,null);
    }

    function task($node,$lower,$upper){
        if ($node == null){
            return true;
        }

        if ($lower !== null && $lower >= $node->val){
            return false;
        }
        if ($upper !== null && $upper <= $node->val){
            return false;
        }

        if (!$this->task($node->left,$lower,$node->val)){
            return false;
        }
        if (!$this->task($node->right,$node->val,$upper)){
            return false;
        }
        return true;
    }

}
```

### 3.2 对称二叉树
```php
101. 对称二叉树
https://leetcode-cn.com/problems/symmetric-tree/

<?php 

class Solution {
   /** 
     * 递归判断，找结束点
     * @param TreeNode $l
     * @param TreeNode $r
     * @return Boolean
     */
    function t($l,$r){
        if ($l == null && $r == null){
            return true;
        }
        if ($l == null || $r ==null){
            return false;
        }
        return ($l->val == $r->val) && $this->t($l->left,$r->right) && $this->t($l->right,$r->left);
    }
    /**
     * @param TreeNode $root
     * @return Boolean
     */
    function isSymmetric($root) {
        return $this->t($root->left,$root->right);
    }
    
}

```

https://leetcode-cn.com/problems/path-sum/
<?php

class Solution {

    /**
     * @param TreeNode $root
     * @param Integer $sum
     * @return Boolean
     */
    function hasPathSum($root, $sum) {
        if($root == null){
            return false;
        }

        $sum = $sum-$root->val;
        if($root->left == null && $root->right == null){
            return ($sum == 0);
        }
        return $this->hasPathSum($root->left,$sum) || $this->hasPathSum($root->right,$sum);
    }
}

```php
113. 路径总和 II
https://leetcode-cn.com/problems/path-sum-ii/
给定一个二叉树和一个目标和，找到所有从根节点到叶子节点路径总和等于给定目标和的路径。

思路：dfs + 回溯

<?php
class Solution {

    public $paths = [];
    /**
     * @param TreeNode $root
     * @param Integer $sum
     * @return Integer[][]
     */
    function pathSum($root, $sum) {
        if($root == null){
            return [];
        }
        
        $this->dfs($root,[],0,$sum);
        return $this->paths;
    }

    function dfs($node,$path,$curSum,$sum){
        if($node == null){
            return ;
        }

        $path[] = $node->val;
        $curSum += $node->val;

       if($node->left == null && $node->right == null && $curSum == $sum){
             $this->paths[] = $path;
         }
        if($node->left != null){
            $this->dfs($node->left,$path,$curSum,$sum);
        }
        if($node->right != null){
            $this->dfs($node->right,$path,$curSum,$sum);
        }
    }

}
```

class Solution {

    /**
     * @param TreeNode $root
     * @return TreeNode
     */
    function invertTree($root) {
        if($root == null){
            return;
        }
        $rightTree = $root->right;
        $root->right = $this->invertTree($root->left);
        $root->left = $this->invertTree($rightTree);
        return $root;
    }
}

104. 二叉树的最大深度

111. 二叉树的最小深度

<?php

class Solution {

    /**
     * 二叉树的最大深度
     * @param TreeNode $root
     * @return Integer
     */
    function maxDepth($root) {
        if($root == null) return 0;
        $maxDepth = 1;
        if($root->left == null && $root->right == null){
            return 1;
        }
        $leftMaxDepth = $this->maxDepth($root->left);
        $rightMaxDepth = $this->maxDepth($root->right);
        return ($leftMaxDepth > $rightMaxDepth) ? $leftMaxDepth + $maxDepth : $rightMaxDepth + $maxDepth;
    }
    
    /**
     * 111. 二叉树的最小深度
     * @param TreeNode $root
     * @return Integer
     */
    function minDepth($root) {
        if($root == null){
            return 0;
        }
        $minDepth = PHP_INT_MAX;
        if($root->left == null && $root->right == null){
            return 1;
        }
        if($root->left !== null){
            $minDepth = min($this->minDepth($root->left),$minDepth);
        }
        if($root->right !== null){
            $minDepth = min($this->minDepth($root->right),$minDepth);
        }
        return $minDepth+1;
    }
    
    
}
