
62. 不同路径 二维数组的：https://leetcode-cn.com/problems/unique-paths

```text

 /**
     * @param Integer $m
     * @param Integer $n
     * @return Integer
     */
    function uniquePaths($m, $n) {
        $dp = [];
        for($i = 1;$i<=$m;$i++){
             $dp[$i][1] = 1;
        }
        for($j = 1;$j<=$n;$j++){
             $dp[1][$j] = 1;
        }
        for($i = 2;$i<=$m;$i++){
            for($j =2 ;$j<=$n;$j++){
                $dp[$i][$j] = $dp[$i-1][$j] + $dp[$i][$j-1];
            }
        }
        
        return $dp[$m][$n];
    }
```
